# Sci

Scientific computation for Elixir.

Work in progress.

Documentation can be found at [https://hexdocs.pm/sci](https://hexdocs.pm/sci).

## Installation

This package can be installed by adding `sci` to your list of dependencies in
`mix.exs`:

```elixir
def deps do
  [
    {:sci, "~> 0.0.1"}
  ]
end
```
